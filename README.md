# SuperQuestioner · 题库练习

在线答题器，支持多题库、多模式、错题本管理、答题分析与统计。

## 部署

将以下文件放到同一目录，推送到 GitHub Pages 即可：

- `index.html` — 页面结构
- `style.css` — 样式
- `app.js` — 逻辑
- `favicon.svg` — 图标
- `*.json` — 题库数据
- `segmentit.js` — 中文分词库（可选，用于主观题评分）
- `*.ogg` — 音效文件（可选）

## 首页功能

首页展示多个卡片入口：

| 卡片      | 说明                  |
| ------- | ------------------- |
| 🔄 重做错题 | 从错题本答题，可切换暂时/长期记忆   |
| 🎯 闯关模式 | 累计错100题失败           |
| ♾ 无限模式  | 答对3次后不再出现，可重置       |
| ⏱ 限时模式  | 自定义时间 + 章节/难度/题型筛选  |
| 📖 预览模式 | 浏览全部题目，支持列表/卡片/滚轮视图 |

顶部右上角按钮：日月图标为\*\*日夜模式切换。

## 添加题库

1. 将 JSON 文件放到同目录。
2. 在 `app.js` 顶部 `QUIZ_SOURCES` 数组添加一行：

```js
const QUIZ_SOURCES = [
  { file: 'Marxism.json', name: '马克思主义基本原理' },
  { file: 'Statistic.json', name: '统计学' },
];
```

1. 进入页面时，各页面会记住上次选中的题库。未选的题库在限时模式下不出现其章节。

### JSON 格式

通用字段：

| 字段           | 必填       | 说明                                   |
| ------------ | -------- | ------------------------------------ |
| `sequence`   | 否        | 题号                                   |
| `chapter`    | 否        | 章节名，缺失时隐藏章节筛选                        |
| `type`       | 否        | 题型，缺失时根据答案推断                         |
| `difficulty` | 否        | `easy` / `medium` / `hard`，缺失时隐藏难度筛选 |
| `question`   | **是**    | 题干文本                                 |
| `answer`     | **是**    | 答案（格式因题型而异）                          |
| `options`    | 选择题**是** | 选项数组                                 |

#### 选择题（`single_choice` / `multiple_choice` / `true_false`）

```json
{
  "sequence": 1,
  "chapter": "第一章",
  "type": "single_choice",
  "difficulty": "easy",
  "question": "题干文本",
  "answer": "A",
  "options": [
    { "label": "A", "text": "选项A" },
    { "label": "B", "text": "选项B" },
    { "label": "C", "text": "选项C" },
    { "label": "D", "text": "选项D" }
  ]
}
```

- `type`：`single_choice` / `multiple_choice` / `true_false`，缺失时根据答案自动推断
- 多选题 `answer` 为多个字母（如 `"ACD"`）
- 判断题无需 `options`，答案为 `正确` 或 `错误`

#### 计算题（`calculation`）

支持多子问，每问可设精确值或范围。支持内嵌表格。

```json
{
  "sequence": 1,
  "chapter": "第一章",
  "type": "calculation",
  "difficulty": "medium",
  "question": "根据以下数据计算：",
  "table": {
    "headers": ["商品", "基期价格", "基期数量", "报告期价格", "报告期数量"],
    "rows": [
      ["甲", "10", "100", "12", "120"],
      ["乙", "8", "150", "9", "180"]
    ]
  },
  "sub_questions": [
    { "id": 1, "text": "第一问：求 X 的值" },
    { "id": 2, "text": "第二问：求 Y 的值" }
  ],
  "answer": {
    "1": { "scope": [0.98, 1] },
    "2": { "value": 3.14 }
  }
}
```

- `table`（可选）— 数据表格，`headers` 为列名数组，`rows` 为数据行数组
- `sub_questions` — 子问题干，`id` 对应 `answer` 的键
- `answer` 字典中，`scope` 表示闭区间范围，`value` 表示精确值（误差 < 0.001）
- 全部答对计一次正确，部分错计一次错误

#### 主观题（`subjective`）

基于中文分词（segmentit）词集对比评分。

```json
{
  "sequence": 1,
  "chapter": "第一章",
  "type": "subjective",
  "difficulty": "hard",
  "question": "结合实际，说明实践是检验真理的唯一标准。",
  "answer": {
    "reference": "实践是检验真理的唯一标准，因为：第一，真理的本性在于主观与客观相符合；第二，实践具有直接现实性的特点，能够把主观认识与客观实际联系起来加以对照；第三，只有实践才能证明认识是否具有真理性。脱离实践的思想理论和逻辑证明都不能作为检验真理的标准。",
    "min_match": 0.25
  }
}
```

- `reference` — 参考答案，分词后与用户答案分词结果对比
- `min_match` — 词重叠率阈值（命中词数 / 参考总词数）
- `segmentit.js` 需放在同目录以启用评分；缺失时降级为简单关键词匹配。

> 向后兼容：也支持旧版 `groups`（分组匹配）和 `keywords`（平铺关键词）格式。

## 预览模式

预览模式用于快速浏览全部题目，不是用来浏览、筛选、查看答案：

**筛选条件**：

- **题库**：多选章节（默认全选），可点「全选」一键全选/取消
- **题型**：单选 / 多选 / 判断 / 计算 / 主观，根据章节自动联动（未选章节时灰色锁定
- 章节未选时，题型区域锁定

**视图切换（右上角按钮）：**

| 视图    | 说明                                       |
| ----- | ---------------------------------------- |
| 📄 卡片 | 逐题浏览，上一题/下一题导航，题干末尾直接显示答案字母              |
| 📋 列表 | 所有题目按列表展示，点击展开查看详情                       |
| ⚙ 滚轮  | 列表模式下出现的开关，启用后题目呈圆筒状滚动，靠近中心的放大展开，远离的缩小收起 |

滚轮模式下：

- 单次滚动前进一题；
- 连续快速滚动后暂停后再滚动时平滑吸附；
- 中心题完全展开，紧邻题半透明；
- 初始定位第一题自动居中。

## 答题模式

| 模式   | 说明                                      |
| ---- | --------------------------------------- |
| 重做错题 | 从错题本抽取，答对移除暂时错题，可选是否含长期记忆               |
| 闯关模式 | 答错100道即失败                               |
| 无限模式 | 答对3次的题不再出现，错题反复出现；顶部显示"已答题 X/∞"；可重置     |
| 限时模式 | 选章节/难度/题型（按章节联动筛选，自定义答题时间（分钟），查看答案时暂停计时 |
| 预览模式 | 浏览全部题目，可筛选章节/题型/题库                      |

### 选项随机排列

每次出现题目时，选项 ABCD 的顺序会被随机打乱（原始 JSON 不变），在渲染层重新编号；判定时以新编号为准。

### 计算/主观题答题流程

- 渲染输入区（计算：每问一个输入框；主观：textarea）
- 用户填写后点击"提交"按钮
- 提交后显示逐问反馈/匹配详情
- 提交按钮隐藏，"下一题"按钮出现，用户手动点击进入下一题（不会自动跳转）

## 错题本

- **暂时错题**：答错自动加入，重做答对后自动移除
- **长期记忆**：用户主动加入 / 写备注时自动加入，重做答对不移除
- 点击错题本可浏览、分类切换（暂时/长期）、移除
- 答错的题在错题本详情页可直接写备注，保存后该题自动从"暂时错题"升为"长期记忆"

### 分析标签页

错题本顶部 chip 切换到蓝色"分析"标签后可见：

- **各章节错题数**（柱状图 · 降序
- **各章节答题正确率**（柱状图 · 与错题数并列显示
- \*\*迟疑最久的题目 Top 10（单位：秒），点击可跳到该题详情
- 页面实时反映当前答题进度
- 右下角"清空分析数据"按钮

## 深色主题

点击顶部右上角 🌙/☀ 切换明暗主题。选择会记住下次打开保持。

## 交互

- 单选/判断：点击即判定，1.5秒自动下一题
- 多选：点击切换选中，确认按钮提交
- 计算/主观：填写后点提交，手动点下一题
- 连击特效：连续3题答对闪绿光，10题掉落+Perfect，可在答题页 `+` 号开关
- 答题点 `+` 号可切换分类、写备注

## 音效

同目录放置以下 `.ogg` 文件即可启用音效（缺失时静默跳过）：

| 文件            | 触发时机             |
| ------------- | ---------------- |
| `orb.ogg`     | 每次答对             |
| `break.ogg`   | 每次答错（同时答题卡四周闪红光） |
| `levelup.ogg` | 累计答对次数达到 10 的倍数时 |

## 备注

- 答题页 `+` 号面板、错题本详情页都可输入备注
- **写了备注的题目自动进入长期记忆**（并退出暂时错题）
- 备注在导出/导入进度 JSON 时会一起带走

## 进度管理

答题页右上角齿轮 ⚙ 可：

- 导出/导入进度 JSON（包含无限模式进度、答题统计、自定义难度、错题本、备注、分析数据）
- 重置全部进度（含无限模式与分析数据）

不点击"新一轮"按钮时，进入无限模式会继承历史已答题数。
